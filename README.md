# Game (Java) — Combat Sandbox

Небольшой Java-проект для практики ООП и чистой архитектуры на примере простой боевой системы.  
Проект создан как учебный и портфельный: акцент на структуре кода, разделении ответственности и тестируемости.

## О проекте

Это не полноценная игра, а компактная “песочница” для отработки:

- объектно-ориентированного проектирования  
- разделения логики и моделей данных  
- базовой боевой механики (урон, взаимодействие сущностей)  
- написания автотестов  

## Структура проекта

.
├─ Main.java # точка входа / демо-запуск
├─ combat/ # боевая логика и правила
├─ model/ # доменные модели (игроки, статы и т.д.)
├─ lib/ # вспомогательные утилиты
├─ tests/ # автотесты
├─ out/ out_tmp/ # артефакты сборки


## Требования

- Java 17+ (рекомендуется)
